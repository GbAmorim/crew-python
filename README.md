Essa aplicação, utiliza o Ollama com api do llama3.2:latest rodando localmente na maquina. Para isso, basta instalar o Ollama e instalar a IA do llama3.2:latest para o computador.
<br>
Foi utilizado a API serper para fazer buscas na web.
<br>
A aplicação basicamente junta prompts de agentes e as tasks passadas pra eles para executar os comandos que foram dados. No caso, formar um texto baseado em um tema dado pelo usuário para uma publicação no Linkedin.
